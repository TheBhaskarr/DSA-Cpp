# Binary Search (DSA in C++)

This folder contains **Binary Search based problems implemented in C++** for data structures and algorithm practice.

Binary Search is one of the most important techniques used to efficiently search in **sorted data structures**. Many interview problems are solved using variations of binary search.

---

## Concepts Covered

- Binary Search Basics  
- Modified Binary Search  
- Binary Search on Answer  
- Searching in Rotated Arrays  
- Peak Finding Problems  
- Binary Search on Matrix  

---

---

## Why Binary Search?

Binary Search reduces the search space by **half in each step**, making it extremely efficient for large datasets.

Example:

- Linear Search → **O(n)**  
- Binary Search → **O(log n)**
